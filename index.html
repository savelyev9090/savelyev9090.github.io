Мои проекты

YouTube Trands Analyser

Telegram-бот, который анализирует комментарии под видео на YouTube и определяет общий эмоциональный настрой аудитории (позитивный, негативный, нейтральный).
Пользователь отправляет ссылку на видео, а бот возвращает статистику настроений и ключевые слова из комментариев.

Стек технологий:

  • Python (Aiogram, Google-Api-Python-Client)
  • SciPy
  • SQLite
  • Docker

RemindMeBot

Telegram-бот-напоминалка с функцией планирования задач и гибким расписанием.
Пользователь может задавать напоминания с повторениями (ежедневно, еженедельно), а бот будет присылать уведомления с возможностью отложить задачу или отметить её выполненной.

Стек технологий:

  • Python (Aiogram)
  • PostgreSQL
  • APScheduler
  • Google-API

BugBlaster

CLI-утилита для автоматического поиска и логирования багов в коде на основе анализа логов и тестов.
Пользователь указывает путь к проекту, а утилита сканирует код, запускает тесты и генерирует отчёт с потенциальными проблемами.

Стек технологий:

  • Python (Click, Pytest)
  • Loguru
  • GitHub Actions
  • JSON
